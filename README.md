# Joke Generator App

This is a simple JavaScript-based application that displays a new random joke every time the page is refreshed.

## Features
- Random joke generation
- Dynamic content update on refresh
- Simple and clean UI

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Use
1. Open index.html in browser
2. Refresh the page to see a new joke
